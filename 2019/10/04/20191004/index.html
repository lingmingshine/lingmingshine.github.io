title: java回炉重造一番
date: 2019-10-01

工作之后在实现某一模块时候，有些东西突然卡壳了，然后去百度一下找到解决方法，再遇见又会忘掉。所以这样的情况很糟糕，说明基础太不扎实了，重新回来理一理基础知识，然后才能继续前行：

Java

1.常量：

整型常量：十进制，八进制（以0开头），十六进制（以0x开头）

实型常量：十进制和科学记数法

布尔类型常量：false和true

字符型（单引号）和字符串型（双引号）

final定义常量的关键字

2.变量：

方法中的变量必须显式地初始化

否则在使用变量的时候就会出错

如果没有显式地初始化，默认状态创建变量会默认值为0

（挺重要的，涉及问题：初始化变量默认值，无法通过本方法打印输出，这样会报错）

成员变量：

全局变量（实例变量）

实现：对象名.变量名 无static修饰 对象被引用，实例变量存在

静态变量（类变量）

实现：对象名.变量名 用static修饰

类名.变量名 生命周期取决于类的生命周期，类被垃圾回收机制彻底回收才会销毁

3.基本数据类型

注意点：范围，正数符号为0放在第一位，负数符号为为1放在第一位

所以byte1字节计算的时候正数第一位为0 所以值为127

而负数第一位为1 所以值为-128

隐式转换（自动类型转换）（automatic type conversion）

显式转换（强制类型转换）

4.运算符

主要在位运算

“>>”右移位运算符 低位舍弃，高位补0

“<<”左移位运算符 高位舍弃，低位补0

按二进制把所有的数字向左或者向右移动

（其实位运算，在大学课程数字逻辑电路和基本组成原理里面涉及得多）

“>>>”无符号右移运算

-6>>>3（当初做面试题遇到过，那时候真的懵啊）

原码：1000 0000 0000 0000 0000 0000 0000 0110

取反加1:

​ 1111 1111 1111 1111 1111 1111 1111 1010

右移3位

​ 0001 1111 1111 1111 1111 1111 1111 1111

​ 结果=536870911

条件运算符

三元运算符

a=b?c:d

b true do c

b false do d
